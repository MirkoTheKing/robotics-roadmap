Updating your GitHub Pages site
A
Web UI (easiest): Open your repo → click index.html → click the pencil ✏ icon → select all (Ctrl+A / Cmd+A) → paste your new code → click Commit changes. Done in under 60 seconds.
B
Git (faster for repeated updates): git pull → replace index.html → git add index.html && git commit -m "update roadmap" && git push. Pages redeploys in ~30s automatically.
C
GitHub CLI: gh repo clone you/robotics-roadmap then same git steps above. Useful if you edit locally in VS Code.
Progress is saved in localStorage — it persists across visits on the same device. It does NOT sync between devices. Use ↺ Reset to start fresh.
